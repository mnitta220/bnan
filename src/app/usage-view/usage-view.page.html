文書表示画面

文書表示画面は、文書が表示されます。


本文／目次

「目次」を選択すると、目次が表示されます。目次には、テキストに見出しとして入力された行が表示されます。

「本文」を選択すると、本文が表示されます。本文として表示されるのは、「目次」で選択中の見出しから、次の見出しの前までです。

たとえば、次のようなテキストが入力されていたとします。

# 第１章 あいうえお
本文かきくけこ。
# 第２章 さしすせそ
本文たちつてと。

この場合、目次には「第１章 あいうえお」と「第２章 さしすせそ」が表示されます。

目次で「第１章 あいうえお」を選択すると、テキスト欄に「本文かきくけこ。」が表示されます。

目次で「第２章 さしすせそ」を選択すると、テキスト欄に「本文たちつてと。」が表示されます。


全表示／黒塗り

「全表示」を選択すると、本文が黒塗りしないで表示されます。

「黒塗り」を選択すると、本文が黒塗りで表示されます。

テキスト欄は、横書き表示なら左右に、縦書き表示なら上下に、指でスクロールすることができます。スクロールバーを動かすことでもスクロールできます。


ツールボタン

〔黒塗り〕
黒塗りの区切りを、次の「/」に移動します。
テキスト欄をタップすることによっても、次に移動します。
〔黒塗り〕
黒塗りの区切りを、前の「/」に移動します。
〔黒塗り〕
黒塗りの区切りを、次の1文字または1単語に移動します。
本文の末尾に移動します。
本文の先頭に移動します。
テキスト欄の高さを大きくします。
適度な高さに調整してください。
テキスト欄の高さを小さくします。

※ 黒塗り表示の時に、テキスト欄をダブルタップすると、黒塗りの区切りが、その位置に移動します。


「更新・変更する」ボタン

「更新・変更する」ボタンを押すと、文書登録画面に遷移します。



白板

「白板(はくばん)」を選択すると、指やタッチペンで手書きができる画面になります。文章を手を動かして書く際に使うことができます。



文章を暗記するコツは、次の５つです。

  1. 文章を適度な長さに分ける。
  2. 文章を声に出して読む。
  3. 文章を手を動かして書く。
  4. 文章を頭の中でイメージ化する。
  5. 覚えるまで何度も繰り返す。
